Data - remote retrieval (pynance.data.retrieve)¶

Wraps Pandas Remote Data Access.

pynance.data.retrieve.equities(country='US')[source]¶

Return a dictionary of equities and the exchange where they are traded.

Currently only US markets are supported

New in version 0.4.0.

Parameters:

country : str, optional

Country code for equities to return, defaults to ‘US’.
Returns:

eq : dict of str

Dictionary mapping equities to a string representing the exchange on which they are traded.

Examples

>>> equities = pn.data.equities('US')
