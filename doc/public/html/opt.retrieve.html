Options - remote retrieval (pynance.opt.retrieve)¶

pynance.opt.retrieve.get(equity)[source]¶

Retrieve all current options chains for given equity.

Changed in version 0.5.0: Eliminate special exception handling.

Parameters:

equity : str

Equity for which to retrieve options data.
Returns:

optdata : Options

All options data for given equity currently available from Yahoo! Finance.

Examples

Basic usage:

>>> fopt = pn.opt.get('f')

To show useful information (expiration dates, stock price, quote time) when retrieving options data, you can chain the call to get() with info():

>>> fopt = pn.opt.get('f').info()
Expirations:
...
Stock: 15.93
Quote time: 2015-03-07 16:00
