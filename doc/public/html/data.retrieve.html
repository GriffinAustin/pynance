Data - remote retrieval (pynance.data.retrieve)¶

Wraps Pandas Remote Data Access.

pynance.data.retrieve.equities(country='US')[source]¶

Return a DataFrame of current US equities.

New in version 0.4.0.

Changed in version 0.5.0: Return a DataFrame

Parameters:

country : str, optional

Country code for equities to return, defaults to ‘US’.
Returns:

eqs : pandas.DataFrame

DataFrame whose index is a list of all current ticker symbols. Columns are ‘Security Name’ (e.g. ‘Zynerba Pharmaceuticals, Inc. - Common Stock’) and ‘Exchange’ (‘NASDAQ’, ‘NYSE’, ‘NYSE MKT’, etc.)

Notes

Currently only US markets are supported.

Examples

>>> eqs = pn.data.equities('US')
